# ⚠️ Axios NPM Supply Chain Attack — CRITICAL

#

Malicious versions axios@1.14.1 and axios@0.30.4 were uploaded to NPM. Library with 100M+ weekly downloads. Attack mechanism: plain-crypto-js@4.2.1 fake dependency injected → cross-platform RAT dropper (macOS/Windows/Linux). It communicates with the C2 server and deletes itself, leaving a clean package.json. Detection: StepSecurity. If you have installed these versions, consider your system as compromised.

  • https://www.stepsecurity.io/blog/axios-compromised-on-npm-malicious-versions-drop-remote-access-trojan

# Claude Code Source Code Leaked from NPM

#

Anthropic accidentally included 500K+ lines of source code and 1000+ files in Claude Code's NPM package. Remarkable details found:

  • Anti-distillation: Poison model copycats by injecting fake tools
  • Undercover mode: Mode that hides AI traces (cannot be turned off)
  • Frustration regex: Detect user swearing with regex
  • Native client attestation: API call validation at Zig level (DRM-like)
  • KAIROS: Unreleased autonomous agent mode (/dream, memory distillation)
  • 250K wasted API calls per day: autoCompact bug (fixed with 3 lines of code)
  • https://alex000kim.com/posts/2026-03-31-claude-code-source-leak/
