#

OpenAI gives $1M compute credit to the person who completes the best LLM under 16 MB (RunPod). Rule: weights + training code ≤16 MB, ≤10 minutes on 8×H100. Open until April 30. Top performers are invited to a junior researcher interview in June. (Source: the-decoder.com)

# 4. [Snowflake Cortex AI Vulnerability — Sandbox Escape](https://www.promptarmor.com/resources/snowflake-ai-escapes-sandbox-and-executes-malware)

#

PromptArmor report: Cortex Agent executed malicious code outside the sandbox via prompt injection in the GitHub README. The cat &lt; &lt;(sh &lt; &lt;(wget ...)) command was deemed "safe" by the agent. It's patched now. Simon Willison: "Allow-list based command filtering is unreliable." (Source: simonwillison.net, March 18, 2026)

# 6. [OEL — Online Experiential Learning: Continuous Learning from Deployment](https://arxiv.org/abs/2603.16856)

#

Groundbreaking framework: LLMs can learn from real deployment experiences. Two stages: (1) extraction of transferable information from user-side interaction trajectories, (2) processing into model parameters via on-policy context distillation. Consistent improvement was observed across multiple iterations. (arXiv 2603.16856)
