# OpenAI $122B Yatırım Turu Kapandı — Değerleme $852B

#

Aylık gelir $2B, yıllık $13.1B ama hâlâ kârsız. ChatGPT 900M+ haftalık aktif kullanıcı, 50M+ abone. Sora kapatıldı, maliyet kontrolleri sıkılaştırılıyor. IPO hazırlıkları sürüyor.

# ⚠️ Axios NPM Supply Chain Saldırısı — KRİTİK

#

axios@1.14.1 ve axios@0.30.4 zararlı versiyonları NPM'e yüklendi. 100M+ haftalık indirmesi olan kütüphane. Saldırı mekanizması: plain-crypto-js@4.2.1 sahte bağımlılık enjekte edildi → cross-platform RAT dropper (macOS/Windows/Linux). C2 sunucusuyla iletişim kurup kendini silip temiz package.json bırakıyor. Tespit: StepSecurity. Bu versiyonları yüklediyseniz sisteminizi ele geçirilmiş kabul edin.

  • https://www.stepsecurity.io/blog/axios-compromised-on-npm-malicious-versions-drop-remote-access-trojan
Bölümler 07

# Claude Code Kaynak Kodu NPM'den Sızdı

#

Anthropic, Claude Code'un NPM paketine yanlışlıkla 500K+ satır kaynak kod ve 1000+ dosya dahil etti. Bulunan dikkat çekici detaylar:

  • Anti-distillation: Sahte tool'lar enjekte ederek model kopyacılarını zehirleme
  • Undercover mode: AI izlerini gizleyen mod (kapatılamaz)
  • Frustration regex: Kullanıcı küfürlerini regex ile tespit
  • Native client attestation: Zig seviyesinde API çağrısı doğrulama (DRM benzeri)
  • KAIROS: Yayınlanmamış otonom agent modu (/dream, memory distillation)
  • Günde 250K boşa API çağrısı: autoCompact hatası (3 satır kodla düzeltildi)
  • https://alex000kim.com/posts/2026-03-31-claude-code-source-leak/

# Google Veo 3.1 Lite — En Ucuz Video Üretimi

#

Google DeepMind en uygun fiyatlı video modeli Veo 3.1 Lite'ı çıkardı. 720p $0.05/sn, 1080p $0.08/sn. 4-8 saniye klip, text-to-video + image-to-video. Sora kapandıktan sonra video alanında rekabet büyük ölçüde Çin'e (Seedance 2.0) kayıyor.
