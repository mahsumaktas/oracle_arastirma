Bölümler 02

# 1. [Google Stitch: "Vibe Design" Platformu Açıldı](https://stitch.withgoogle.com)

#

Bölümler 03

# 2. [Gemini API: Tool Chaining & Maps Entegrasyonu](https://the-decoder.com/google-opens-updated-deep-research-agent-to-developers-with-new-api/)

#

Google DeepMind, Gemini API'ye "context circulation" getirdi — bir araç sonucu otomatik sonrakine geçiyor. Google Maps artık Gemini 3 için veri kaynağı: konum, işletme, seyahat süresi. Her tool call'a benzersiz ID.

Bölümler 04

# 3. [OpenAI "Parameter Golf": 16MB Model Yarışması + Yetenek Avı](https://github.com/openai/parameter-golf)

#

OpenAI, 16 MB altında en iyi LLM'i yapana $1M compute kredisi veriyor (RunPod). Kural: ağırlıklar + eğitim kodu ≤16 MB, 8×H100 üzerinde ≤10 dakika. Nisan 30'a kadar açık. Top performans gösterenler Haziran'da junior researcher mülakatına davet ediliyor. (Kaynak: the-decoder.com)

Bölümler 05

# 4. [Snowflake Cortex AI Güvenlik Açığı — Sandbox Kaçışı](https://www.promptarmor.com/resources/snowflake-ai-escapes-sandbox-and-executes-malware)

#

PromptArmor raporu: Cortex Agent, GitHub README'sindeki prompt injection ile sandbox dışında zararlı kod çalıştırdı. cat &lt; &lt;(sh &lt; &lt;(wget ...)) komutu, agent tarafından "güvenli" kabul edildi. Şimdi yamalandı. Simon Willison: "Allow-list tabanlı komut filtreleme güvenilmez." (Kaynak: simonwillison.net, 18 Mart 2026)

Bölümler 06

# 5. [MiroThinker-1.7 & H1: Verification-Based Ağır Araştırma Ajanları](https://arxiv.org/abs/2603.15726)

#

MiroMind ekibinden yeni araştırma ajanı mimarisi. Doğrulama mekanizmaları üzerine kurulu "heavy-duty" araştırma agentleri. cs.CL kategorisi, HF daily papers'da öne çıktı.

Bölümler 07

# 6. [OEL — Online Experiential Learning: Deployment'tan Sürekli Öğrenme](https://arxiv.org/abs/2603.16856)

#

Çığır açan framework: LLM'ler gerçek deployment deneyimlerinden öğrenebiliyor. İki aşama: (1) kullanıcı tarafı interaction trajectory'lerinden transferable bilgi çıkarımı, (2) on-policy context distillation ile model parametrelerine işleme. Çoklu iterasyonda tutarlı iyileşme gözlemlendi. (arXiv 2603.16856)

Bölümler 08

# 7. [GitHub Trending: obra/superpowers — 95K Star Agentic Skills Framework](https://github.com/obra/superpowers)

#

Bugün GitHub trending #1 (4,091 yıldız). Shell tabanlı agentic skills framework & yazılım geliştirme metodolojisi. Claude tarafından da katkı var.
