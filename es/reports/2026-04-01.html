# ⚠️ Ataque a la cadena de suministro de Axios NPM: CRÍTICO

#

Las versiones maliciosas axios@1.14.1 y axios@0.30.4 se cargaron en NPM. Biblioteca con más de 100 millones de descargas semanales. Mecanismo de ataque: plain-crypto-js@4.2.1 dependencia falsa inyectada → dropper RAT multiplataforma (macOS/Windows/Linux). Se comunica con el servidor C2 y se elimina, dejando un paquete.json limpio. Detección: StepSecurity. Si ha instalado estas versiones, considere que su sistema está comprometido.

  • https://www.stepsecurity.io/blog/axios-compromised-on-npm-malicious-versions-drop-remote-access-trojan
Secciones 07

# Código fuente de Claude Code filtrado de NPM

#

Anthropic incluyó accidentalmente más de 500.000 líneas de código fuente y más de 1000 archivos en el paquete NPM de Claude Code. Detalles destacables encontrados:

  • Anti-destilación: Envenenar a los imitadores del modelo inyectando herramientas falsas
  • Modo encubierto: Modo que oculta rastros de IA (no se puede desactivar)
  • Frustración expresión regular: Detectar usuario que dice malas palabras con expresión regular
  • Certificación de cliente nativo: Validación de llamadas API a nivel Zig (similar a DRM)
  • KAIROS: Modo agente autónomo inédito (/sueño, destilación de memoria)
  • 250.000 llamadas API desperdiciadas por día: error de autoCompact (solucionado con 3 líneas de código)
  • https://alex000kim.com/posts/2026-03-31-claude-code-source-leak/
